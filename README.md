## Getting Started

This project require NodeJS v12.

### Install
Clone this project

```bash
> git clone https://github.com/Hendra088/HENDRA03

> cd GaroxBot
```

Install the dependencies:

```bash
> npm install 
> npm install gify-cli -g
```

### Usage
Run the Whatsapp bot

```bash
> npm start
```

after running it you need to scan the qr
